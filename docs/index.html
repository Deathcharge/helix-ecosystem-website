🚀 Quick Start

Installation

Install any package from PyPI:

pip install routine-engine ucf-protocol helix-chat-engine

Basic Example

from helix_chat_engine import ChatServer
from routine_engine import RoutineEngine
from ucf_protocol import UCFProtocol

# Initialize components
chat = ChatServer()
engine = RoutineEngine()
ucf = UCFProtocol()

# Run integrated system
chat.run(host="0.0.0.0", port=8000)

🔌 Integration & Deployment

Standalone Usage

Each package works independently with zero external dependencies:

pip install helix-chat-engine
from helix_chat_engine import ChatServer
server = ChatServer()
server.run()

Integrated Ecosystem

Or combine multiple packages for powerful integrated systems:

from helix_integration import HelixEcosystem

ecosystem = HelixEcosystem()
ecosystem.add_chat_engine()
ecosystem.add_llm_orchestration()
ecosystem.add_workflow_automation()
ecosystem.start()
